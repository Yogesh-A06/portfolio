Project 03

AQ-Nexus

AQ-Nexus is an end-to-end full-stack machine learning platform designed to forecast air quality using advanced deep learning and graph-based techniques. It leverages Bi-LSTM models for time-series forecasting and Graph Neural Networks (GNNs) to capture spatial dependencies across environmental data. The system analyzes historical weather and pollution metrics to accurately predict next-day NO₂ and O₃ levels. It also features a real-time monitoring dashboard and a predictive analytics engine to support public health decisions.
